# CloudSense Lite

CloudSense Lite is an AI-powered cloud cost analysis tool that helps solo AWS users
understand where their cloud spend is going and what to fix first.


## MVP Goal
Upload an AWS Cost & Usage Report (CSV) and receive:
- Service-wise cost breakdown   
- Monthly spending trend
- 1â€“2 actionable cost insights


## Non-Goals (for MVP)
- No multi-cloud support
- No real-time monitoring
- No automatic cost changes

## Tech Stack (Initial)
- Backend: Python + FastAPI + Pandas  
- Frontend: React
- AI: OpenAI (analysis + insights)
